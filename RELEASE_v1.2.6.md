## Installation

Download the latest version and make it executable:

```bash
chmod +x underscorify.sh
```

## Usage

```bash
# Basic usage
./underscorify.sh "filename with spaces.txt"

# Process multiple files
ls *.txt | ./underscorify.sh

# Allow hidden files
./underscorify.sh --hidden ".hidden file.txt"
```
